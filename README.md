# Stock Price Prediction Using LSTM, RNN and Ensemble Learning

This repository contains a machine learning system for predicting stock prices
using deep learning models, including LSTM and RNN architectures, combined
through an ensemble learning strategy.

The project focuses on improving prediction accuracy and training efficiency
by leveraging the Ray framework for scalable and parallel model training.

---

## Project Overview

The objective of this project is to build a robust forecasting system for stock
market data using deep learning and ensemble techniques.

Multiple sequence models are trained and combined to improve the stability and
accuracy of predictions compared to a single-model approach.

---

## Key Features

- Stock price forecasting using deep learning
- LSTM and RNN based sequence models
- Ensemble learning for improved prediction performance
- Distributed and parallel training using the Ray framework
- End-to-end pipeline from data preparation to prediction

---

## Modeling Approach

The system consists of:

- Data preprocessing and time-series window generation
- LSTM-based prediction model
- RNN-based prediction model
- Ensemble strategy to combine predictions from multiple models
- Model evaluation and comparison

---

## System Architecture

1. Data loading and preprocessing  
2. Time-series feature construction  
3. Training LSTM and RNN models  
4. Distributed training using Ray  
5. Ensemble prediction generation  
6. Model evaluation
